## ✨ Features

- 🧠 **AI Chat** — Smart replies in a dedicated channel  
- 🔄 **Multi‑Model** — Switch between Groq models anytime  
- 🎨 **Beautiful Embeds** — Stylish purple‑themed responses  
- 📝 **Custom Prompts** — Set the AI’s personality  
- 🛡️ **Staff Controls** — Admin commands locked to a role  
- ⚡ **Auto‑Retry** — Handles rate limits gracefully  
- 💾 **Persistent** — Settings save automatically
